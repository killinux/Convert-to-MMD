# 编辑映射功能 - 快速开始

## 🎉 功能已完成！

编辑骨骼映射功能（Method B: 完整 MMD 骨骼列表）已成功实现并同步到 Blender 插件目录。

## 快速测试步骤

### 1️⃣ 重启 Blender
```
1. 完全关闭 Blender（确保所有进程都已终止）
2. 重新打开 Blender
3. 确保 "XPS to PMX" 插件已启用
```

### 2️⃣ 加载 XPS 模型
```
1. File → Import → XPS Model (.xps)
2. 选择一个包含手指骨的 XPS 模型
3. 选中导入的骨架
```

### 3️⃣ 运行自动映射
```
1. 右侧面板 → "XPS to PMX Mapper" 标签
2. 点击 "① AUTO DETECTION" 区域
3. 点击 "🔄 Auto Map Bones" 按钮
4. 等待映射完成（应看到"Mapped 177/177 bones"）
```

### 4️⃣ 打开树形结构面板
```
1. 右侧面板 → "XPS to PMX Mapper" 标签
2. 向下滚动找到 "🌳 MMD 骨骼树形结构" 面板
3. 点击面板标题展开
```

### 5️⃣ 测试编辑功能
```
方式 A：直接在树形结构中编辑
1. 在树形结构中搜索"左手首"或"センター"
2. 点击那个骨骼名称
3. 在右侧"骨骼详情"面板中，点击"✎ 编辑映射"按钮
4. 编辑映射对话框会弹出
5. 从下拉列表中选择新的 MMD 骨骼
6. 点击"✓ 确定"应用修改

方式 B：通过问题列表编辑
1. 打开"⚠️ 未映射骨骼检查"面板
2. 查看问题骨骼列表
3. 点击一个问题骨骼
4. 在"骨骼详情"面板中点击"✎ 编辑映射"
5. 选择新的映射并确认
```

## 手指骨修正示例

这是最常见的需要修正的映射类型：

### 问题描述
自动映射可能将手指骨（如 lThumb1）映射到错误位置（如 センター）

### 修正步骤

```
1. 搜索：在树形结构的搜索框输入"左手"

2. 展开：找到"左手首"骨骼，展开它看子骨骼

3. 检查：查看映射状态，如果显示"lThumb1"或其他错误映射

4. 编辑：
   - 点击"左手首"
   - 点击"✎ 编辑映射"
   - 选择正确的 MMD 骨骼（通常就是"左手首"本身或其子骨骼）
   - 点击"✓ 确定"

5. 验证：
   - 树形结构应更新，显示新的映射
   - 检查统计信息，覆盖率应提高
   - 在"未映射骨骼检查"中，该骨骼应消失
```

## 提示和技巧

### 💡 使用搜索功能
- 在树形结构的搜索框输入骨骼名字的一部分
- 例如：搜索"手"会显示所有包含"手"的骨骼

### 💡 批量修改相同的骨骼
- 如果多个 XPS 骨骼需要映射到同一个 MMD 位置
- 依次编辑每个，重复相同的选择过程
- 系统会自动处理冲突

### 💡 检查父级关系
- 编辑后，新映射的父级关系会自动验证
- 如果出现"⚠️ 父级不匹配"警告，检查新映射是否合理
- 通常自动检测会正确处理这个问题

### 💡 查看统计信息
- "📊 统计摘要"显示整体的映射情况
- XPS 映射覆盖率：反映有多少 XPS 骨骼已成功映射
- MMD 使用率：反映有多少 MMD 骨骼被使用
