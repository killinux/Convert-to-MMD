## 🚀 如何使用新的树形面板

### 第一步：重启 Blender
1. **完全关闭 Blender**（如果打开了）
2. 重新打开 Blender
3. 加载你的 XPS 模型

### 第二步：使用自动映射
1. 在 3D 视口右侧找到 "XPS to PMX Mapper" 面板
2. 点击 **"Auto Detection"** 中的 "Auto Map Bones" 按钮
3. 等待映射完成（应该显示 "Mapped 177 bones" 等）

### 第三步：查看新的树形面板
现在你应该看到一个新的面板：

```
🌳 MMD 骨骼树形结构
```

这是一个**友好的树形结构展示**，显示完整的 MMD 骨骼层级。

## 📊 树形显示的各种元素说明

### 展开/折叠符号
```
▼  表示该骨骼已展开，点击可折叠
►  表示该骨骼已折叠，点击可展开
-  表示该骨骼无子骨骼
```

### 权重星标
```
⬜⬜⬜⬜⬜ - 权重很大 (> 5%)
⬜⬜⬜⬜   - 权重较大 (2-5%)
⬜⬜⬜     - 权重中等 (0.5-2%)
⬜⬜      - 权重较小 (< 0.5%)
⭕       - 无权重
```

### 骨骼类型图标
```
🔴 控制骨 (Control)
🟢 脊柱骨 (Spine)
🔵 四肢骨 (Arm/Leg)
🟡 D-骨 (D-Bone)
🟣 IK链 (IK)
🌀 扭转骨 (Twist)
```

### 状态图标
```
✓ 绿色复选 - 正确映射
⚠️ 黄色警告 - 父级不匹配或信息不完整
❌ 红色叉  - 未映射或无效引用
```

## 🔧 技术实现细节

### 文件结构
```
xps_to_pmx/
├── __init__.py (已更新，导入 mmd_bone_tree_ui)
├── mmd_bone_tree_ui.py (新增，树形 UI 逻辑)
├── mapping_ui.py (原有，其他面板)
└── mapping/
    └── presets/
        └── mmd_standard_skeleton.json (树形数据源)
```

### 核心类

**XPSPMXBoneTreeProperties**
- 存储 UI 状态（搜索词、过滤选项、显示模式）

**XPSPMX_OT_toggle_bone_expand**
- Operator：展开/折叠骨骼

**XPSPMX_OT_select_bone_detail**
- Operator：选中骨骼显示详情

**XPSPMX_PT_mmd_bone_tree**
- 主面板：树形结构显示

**XPSPMX_PT_bone_detail_panel**
- 右侧面板：骨骼详情

### 数据流
```
mmd_standard_skeleton.json (MMD标准骨骼库)
         ↓
config.bone_mappings (XPS映射)
         ↓
树形面板 (可视化)
         ↓
用户交互 (搜索、编辑、选择)
```
